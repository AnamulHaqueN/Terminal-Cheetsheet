### Shortcut for vs code to terminal 

***Toggle terminal:***

Switch between editor and terminal
From editor → terminal
```ts
Ctrl + `
```
Press again to go back to the editor.

***From terminal → editor:***
```ts
Ctrl + 1
```

***switch editor groups:***
```ts
Ctrl + 2, Ctrl + 3
```

***Open terminal directly:***
```ts
Ctrl + Shift + `
````

***Extra useful terminal shortcuts:***

| Action            | Shortcut           |
| ----------------- | ------------------ |
| New terminal      | `Ctrl + Shift + `` |
| Close terminal    | `Ctrl + Shift + W` |
| Next terminal     | `Ctrl + PageDown`  |
| Previous terminal | `Ctrl + PageUp`    |
